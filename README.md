## 🔧 Workflow Steps


### 1. Customer Intake
Collect, validate and normalize positions from SWIFT Gateway; attach a runId and timestamp for traceability.

### 2. Retrieve
Execute retrieve phase for the RAG pattern: persist interim state, enforce guardrails, and emit structured JSON results.

### 3. Ground
Execute ground phase for the RAG pattern: persist interim state, enforce guardrails, and emit structured JSON results.

### 4. Sanctions Screening

### 5. Limit Control
